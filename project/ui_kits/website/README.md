# UK Barre Academy — Website UI Kit

A high-fidelity recreation of the UK Barre Academy marketing site. Premium, editorial, calm.

## Files
- `index.html` — full clickable prototype (hero → method → programme → cohorts → application).
- `Nav.jsx` — top nav with bone/ink behaviour over scroll
- `Hero.jsx` — full-bleed editorial hero
- `Method.jsx` — three-column principle block
- `Programme.jsx` — module list, monospace indices
- `Cohorts.jsx` — upcoming intakes
- `Testimonial.jsx` — single quiet quote, no headshot
- `Apply.jsx` — application strip with dark surface
- `Footer.jsx` — quiet 4-col footer

## Notes
- Imagery uses **placeholder bone blocks** with subtle copy because the brief did not include licensed photography. Replace with real studio shots when available.
- All colour, type, spacing, and motion comes from `/colors_and_type.css`.
- Navigation states are click-through but non-functional (no real backend).
