If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick start
- Drop in `colors_and_type.css` for tokens.
- Reference `assets/UBA-logo-black.png` (light) or `assets/UBA-logo-white.png` (dark).
- The UI kit at `ui_kits/website/` is the source of truth for component look-and-feel.
- Key vibe: premium, editorial, calm. Closer to high-end health tech than fitness. No emoji. No exclamation marks. No bright colour. No "fitness" energy.
